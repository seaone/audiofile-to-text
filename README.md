### Flask Example

```sh
$ python -m venv venv
```

```sh
$ source ./venv/bin/activate
```

```sh
$ pip install -r ./requirements.txt
```

```sh
$ export FLASK_ENV=development
```

```sh
$ flask run
```

go to http://localhost:5000
